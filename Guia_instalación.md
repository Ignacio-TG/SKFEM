## Guía de instalación SKFEM
1. Clonar el repositorio que contiene la fork de SKFEM: [Link Repositorio](https://github.com/Ignacio-TG/SKFEM.git).

2.	Abrir la terminal en la ruta de la carpeta donde se clonó el repositorio.

3.	(Opcional) Crear un environment para evitar conflictos de dependencias y tener una instalación más limpia. Debe ser Python 3.10 o superior.

4.	Instalar la fork de skfem con el siguiente comando: 

```cmd
pip install -e .[dev]
```
